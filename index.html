Computer struct

Monitor port

This is used to connect comp to monitor

Headphone port

This is used to connect comp to headphone to hear things

Mic port

This is used to connect comp to mic to make others/comp hear you :)

Mouse

Thwe mouse controls the cursor on the screen.

Keyboard

This is used to type things in.

HHD

This is used to store data. Typically, HHD can store lots of long term stuff.

SSD

This is used to store data. SSD can store lots of long term stuff or it might be used as a secondary drive for doing some tasks quickly.
